### 开源微商城，可供商用或学习

### 技术架构

项目使用spring boot

数据持久层 hibernate

安全框架 shiro

连接池 druid

前端 vue route weui jquery bootstrap



### 使用说明

引入第三方jar包 src/main/webapp/WEB-INF/lib

本项目使用vue分离了前后端，并没有使用模版引擎，运行项目以后，打开静态文件即可

前端链接 http://localhost:8080/home/index.html

后台链接 http://localhost:8080/admin/index.html
